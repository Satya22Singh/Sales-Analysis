Sales Data Analysis using Python
 

This project focuses on analyzing a sales dataset using core Python without external libraries. The goal is to extract meaningful business insights such as revenue trends, customer behavior, and product performance.

**Technologies Used**
Python
File Handling
Lists & Dictionaries
Datetime Module

File: sales_data.csv
**Columns:**
Order ID
Date
Customer Name
Product
Category
Quantity
Price
**Problems solved in this project after analysis**
What is the total revenue generated from all sales?
How many total orders were placed?
What is the total number of items sold?
What is the total revenue generated by each category?
What is the total revenue generated by each product?
Which product has the highest quantity sold?
Which customer spent the most money?
What is the total revenue generated on each date?
What is the total revenue between two given dates?
How to filter and export all orders under the "Electronics" category into a new CSV file?
